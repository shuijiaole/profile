# 头像图片说明

请将你的头像图片重命名为 `avatar.jpg` (或 `avatar.png`) 并放置在此 `public` 文件夹中。

## 支持的图片格式

- avatar.jpg
- avatar.png
- avatar.webp

## 图片要求

- **推荐尺寸**: 500x500 像素或更大（正方形）
- **推荐格式**: JPG 或 PNG
- **文件大小**: 建议小于 500KB

## 使用方法

1. 将你的头像图片复制到 `d:\jl\portfolio\public\` 目录
2. 确保文件名为 `avatar.jpg` 或 `avatar.png`
3. 刷新浏览器即可看到效果

如果使用其他文件名或格式，需要修改 `src/components/sections/hero.tsx` 中的 Image 组件 src 属性。
